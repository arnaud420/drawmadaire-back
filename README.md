# Drawmadaire Back

## Getting Started

```
git clone https://gitlab.com/drawmadaire/drawmadaire-back
cd drawmadaire-back
touch .env

DATABASE_HOST=host
DATABASE_PORT=port
DATABASE_USERNAME=username
DATABASE_PASSWORD=password
DATABASE_DATABASE=database
JWT_SECRET=secret
FRONT_URL=http://localhost:3000
API_URL=http://localhost:3080
SENGRID_API_KEY=key
```
